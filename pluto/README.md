# Pluto

The Synnax React component library.

## Installation

### NPM

```bash
npm install @synnaxlabs/pluto
```

### Yarn

```bash
yarn add @synnaxlabs/pluto
```
### PNPM

```bash
pnpm add @synnaxlabs/pluto
```
