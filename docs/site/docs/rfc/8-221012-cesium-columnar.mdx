**Start Date** - 2022-10-12 <br />
**Authors** - emilbon99 <br />

# 0 - Summary

In this RFC I propose a redesign of Cesium's architecture in order to support
irregular time series data. This is a major change, but attempts to maintain
a similar API and design to the current version of Cesium.
