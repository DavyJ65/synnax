# Get Started

## Start a Cluster

Start a single node cluster for learning.

## Acquire Data

Configure channels and start reading data from a file, or use your programming language
choice to write directly to a cluster.

## Analyze Telemetry

Quickly retrieve, analyze, and post-process large quantities of relevant data.

## Create Visualizations

Use the Synnax UI to visualize and explore a cluster's telemetry.
