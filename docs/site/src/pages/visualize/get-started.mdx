This page will guide you through the basics of installing and using the Synnax managment
and visualization UI, Delta.

## Overview

The Synnax Delta UI allows you to manage a cluster and visualize its telemetry. Use it

1. To visualize channel data.
2. In conjunction with a client library to filter relevant data for post processing.
3. To observe cluster metrics and system health.
4. Configure channels and manage cluster resources.

## Install

Install Delta by downloading the latest release for your operating system.
